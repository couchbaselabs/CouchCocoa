Detailed Description

A convenience class that glues Couchbase Mobile into CouchCocoa.

On creation, starts up an instance of Couchbase (or CouchbaseMobile). This object will have a placeholder URL until the embedded server has started up, so you can't access it (i.e. creating any databases) until then.
